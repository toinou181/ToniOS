## 🚀 UTILISATION

### Étape 1 : Accès Initial
1. Ouvrir `admin_dashboard.html` dans un navigateur
2. Saisir les identifiants depuis `.admin_auth`
3. Passer l'authentification sécurisée

### Étape 2 : Navigation
- **Onglets supérieurs** : Basculer entre fichiers
- **Explorateur latéral** : Sélectionner fichiers système
- **Terminal** : Exécuter des commandes

### Étape 3 : Administration
- **Modifier le code** directement dans l'éditeur
- **Sauvegarder** avec Ctrl+S ou bouton
- **Surveiller** les logs en temps réel
