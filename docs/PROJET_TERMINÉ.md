## 🚀 **FONCTIONNALITÉS PRINCIPALES**

### 🔑 **ACCÈS ADMIN**
1. Connectez-vous avec n'importe quel utilisateur valide
2. Dans le champ mot de passe, tapez l'un des codes :
   - `ADMIN_TONIOS_2025`
   - `MASTER_ACCESS_CODE`
   - `ULTRA_SECURE_ADMIN`
3. Le dashboard admin s'ouvrira automatiquement dans une nouvelle fenêtre sécurisée
